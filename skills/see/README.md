# SEE Skill

S.EE platform API integration for short URL management, text sharing, and file sharing.

## Setup

1. Copy `.env.example` to `.env`:

   ```bash
   cp .env.example .env
   ```

2. Edit `.env` and fill in your S.EE API key (get it from https://s.ee/user/developers/):

   ```
   SEE_API_KEY=your_actual_api_key
   ```

## Install

Copy the `see/` directory to `~/.claude/skills/`:

```bash
cp -r see ~/.claude/skills/
```
